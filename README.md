# Smart Realty — Real Estate Price Prediction

Интеллектуальная система прогноза цен на недвижимость на основе машинного обучения.

## 📋 Описание

Smart Realty использует модель линейной регрессии для прогноза цен жилья в зависимости от следующих факторов:
- 🏢 Возраст дома (лет)
- 🚇 Расстояние до ближайшей станции метро (метры)
- 🏪 Количество магазинов рядом
- 📍 Средняя цена в районе

## 🎯 Функционал

- **Веб-интерфейс**: Интерактивный HTML интерфейс для прогноза цен
- **График зависимостей**: Визуализация влияния расстояния до метро на цену
- **Данные недвижимости**: Обучение на реальных данных Тайваньского рынка недвижимости
- **API**: Python скрипты для тестирования и обучения модели

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- pip или conda

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/your-username/realestate_ai.git
cd realestate_ai

# Создайте виртуальную окружение
python -m venv .venv
source .venv/bin/activate  # На Windows: .venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt
```

### Обучение модели

```bash
python train.py
```

Модель будет обучена и сохранена в файл `re_model.pkl`.

### Интерактивное тестирование

```bash
python test.py
```

Введите параметры и получите прогноз цены.

### Запуск веб-интерфейса

```bash
cd templates
python -m http.server 8000
```

Откройте браузер и перейдите на `http://localhost:8000/index.html`

## 📊 Данные

Датасет содержит информацию о ценах на жилье в Тайване с параметрами:
- X1: Дата сделки
- X2: Возраст дома
- X3: Расстояние до метро
- X4: Количество магазинов рядом
- X5: Широта
- X6: Долгота
- Y: Цена за единицу площади

## 🔧 Технологии

## 📈 Метрики модели

Модель использует коэффициент **R²** для оценки качества прогноза.
